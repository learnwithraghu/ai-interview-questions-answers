# Question 2: The Role of Temperature

**Question:** What is the role of temperature in LLM generation?

**Answer:**

### 1. Concept Explanation
Temperature is a hyperparameter that controls the randomness and creativity of the LLM's output. It works by scaling the logits before they pass through a softmax function. 
*   **Low Temperature (e.g., 0.0 - 0.3):** Flattens the distribution towards the highest-probability tokens. The output becomes deterministic, repetitive, and focused.
*   **High Temperature (e.g., 0.7 - 1.0+):** Flattens the probability distribution, giving lower-probability tokens a higher chance of being selected. This produces more diverse and creative text.

### 2. Real-World Example
If you are building an AI agent that extracts structured JSON data from invoices, you would set the temperature to `0.0`. You want strict, predictable output with zero hallucination. If you are building an AI marketing tool to brainstorm hundreds of unique blog post titles, you would set the temperature closer to `0.8` to encourage variety.

### 3. This is how I would answer this
"Temperature controls the randomness of the model's predictions. If I'm building something that requires strict accuracy—like extracting JSON from a document or writing code—I set the temperature to 0 so the model always picks the most statistically likely tokens. But if I'm building a creative writing assistant and need diverse brainstorming, I'll bump the temperature up to 0.7 or higher to allow the model to take more creative risks."
