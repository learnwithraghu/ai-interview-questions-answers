Thought: reason about what information is missing.Action: choose a predefined tool to call.Observation: receive the real tool result.Loop: repeat until enough evidence exists.Agent core: this powers many practical tool-using systems in production workflows.Stop condition: the loop ends when the model has enough evidence to answer confidently.

Grounding: tools replace guesses with observations.Search: fetch real-time facts instead of hallucinating.Calculator: compute arithmetic instead of approximating.Evidence: each observation guides the next thought.Orchestration: the model chooses tool and arguments based on its reasoning trace deliberately.Reliability: observations reduce hallucination on factual, computational, database, or real-time tasks.

Answer

Say it

"ReAct stands for Reasoning and Acting. It's a prompting framework where the LLM thinks, takes an action using an external tool, observes the result, and loops until solved."
